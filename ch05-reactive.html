Reactive Programming

Reactive programming is a way of programming that lets you specify how data flows through user interface components; once this is set up, the DOM updating is handled for you automagically. (As noted in #chapter02, this is what all the Cool Kids™ are using.) In this chapter, you will write études that use three ClojureScript libraries that interface with Facebook®’s React JavaScript library:

  • Quiescent
  • Reagent
  • Om
